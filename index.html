Selected Projects

ESP8266 Environmental Data Logger

Environmental monitoring device designed for long-term, unattended data collection with minimal power consumption.

The system supports dual operation modes: a low-power battery-powered mode using deep sleep for periodic measurements, and a continuous 5V-powered mode that enables Wi-Fi access and a built-in web interface.

  • Temperature, barometric pressure, and soil humidity sensing
  • Low-power operation with deep sleep (~10 days battery life)
  • Local SD card data logging
  • HTML web interface hosted on ESP8266 access point
  • Modular, serviceable design

Technologies: ESP8266, Arduino C++, HTML, SD Card, Sensors
